# To-Do
A simple to-do list app made with React. Items are saved to `localStorage` using the [Web Storage API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Storage_API).

## How to run
* Clone this project: `git@github.com:bgrapes/to-do.git`
* Run `yarn && yarn start`
